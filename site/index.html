Not a keyboard tester

A fault investigator for real switch behavior.

KeySurgeon is built around evidence the tool can honestly collect: chatter, missed presses, suspicious holds, extra events, and saved test results.

Compare it to a browser keyboard tester.

ChatterFinds double-fires that ordinary key-light testers miss.
Watch modeRuns while you type normally and catches chatter in the act.
ReportRolls the latest evidence into board health and next action.

Replacement last

The fix path stays honest.

Every ladder starts with cheap, reversible steps. Replacing the board is only shown as the last rung.

1filterblock bounce now
2airclear debris
3cleancontact and stem
4swaphot-swap switch
5solderreplace switch
6replacelast resort

Local by design

No typed text stored, no telemetry, no mystery upload.

KeySurgeon observes key labels and timing-derived verdicts. Runtime JSON stays under your local user data folder unless you set KEYSURGEON_HOME, and --plain keeps output scriptable.

HEALTHYtested behavior looked normal
WATCHsuspicious; retest or monitor
DEGRADINGclean or isolate the switch
FAILINGrepeatable fault; follow the ladder
